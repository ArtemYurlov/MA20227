## How to add things:

* `git pull upstream master`
* `git add [files]`
* `git commit -m "Change things"` (present tense, make it helpful!)
* Repeat `add` and `commit` - little and often
* `git pull upstream master`
* `git push origin master`
* On your GitHub repo: `Create pull request`
